# Gurdwara Sahib Switzerland 🪯

Welcome to the official website of **Gurdwara Sahib Switzerland**!

This platform serves as a digital gateway to our community in Langenthal, providing information about our history, daily Diwan schedules, educational programs, and selfless services (Sewa).

## ✨ Features
- **Majestic Animations**: High-performance Framer Motion transitions designed for a premium feel.
- **Multilingual Support**: Information available in English and Punjabi.
- **Optimized for Performance**: Built with Next.js 15, featuring hardware-accelerated elements and a sub-2.5s LCP.
- **Vercel Ready**: Static site generation (SSG) for lightning-fast page loads.

## 🚀 Tech Stack
- [Next.js](https://nextjs.org/) (App Router)
- [Framer Motion](https://www.framer.com/motion/) (Optimized with LazyMotion)
- [Lucide React](https://lucide.dev/) (Icons)
- [Tailwind CSS](https://tailwindcss.com/)
